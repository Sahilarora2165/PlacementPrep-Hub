C Campus Compass

Mission

Built so useful campus knowledge does not disappear.

Campus Compass turns scattered student advice into searchable guides, saved posts, discussions, reports, and living local stats.

About the platform

Campus Compass collects student-written guides for placements, projects, exams, events, campus life, travel, and technical learning. Every account, post, like, comment, report, saved item, and message is stored in this browser with localStorage.

Its purpose is simple: help students publish practical information once, then let others save it, discuss it, report outdated details, and reuse it when they need it.
